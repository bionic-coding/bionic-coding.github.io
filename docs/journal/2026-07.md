## [2026-07-13 13:30] learning | Soft-launch retrospective — ADR-0001-0004 in hindsight

Set out to ship the soft launch — the manifesto site, the Lessons section, and the first post — on a deliberately boring, reproducible stack (Jekyll 4.4 + pinned Ruby, GitHub Pages via Actions).

What didn't land cleanly: the custom presentation layer over Minima (ADR-0003) shipped before the visual design was actually settled. The launch is trailed in git by a run of "color fix" → "contrast" → "background color cleanup" commits — the theme went live and then needed several contrast/color passes in production. That should have been a pre-launch pass, not a post-launch scramble.

What surprised me: the taxonomy wasn't as settled as the plan assumed — "AI Literacy" got renamed to "Lessons" immediately after launch. A label that read fine in the content brief didn't survive contact with the live nav.

Biggest realization: ADR-0004 (single-source the homepage/content from research) was aspirational until today. It only became real when the GPT-5.6 / Opus 4.8 / Fable ingests gave the site cited primary sources to draw on — and even then, the OpenAI blog was too large to fetch and had to come in as a downloaded system-card PDF. Next time: stand up the research pipeline *before* the content depends on it, so drafts aren't blocked waiting for sources.
